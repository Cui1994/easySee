# 直播通 easySee
一款跨平台主播搜索关注的web应用
## 现有功能：
+ 平台+房间号主播关注
+ 搜索主播（数据库及网络搜索，网络搜暂不支持全民和虎牙直播）
+ Hot主播显示
+ 邮箱提醒服务
